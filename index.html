JMott Collage Maker

A lightweight, offline desktop application for creating clean, printable photo collages — without subscriptions, cloud uploads, or AI.

What it does

  • Create photo collages using predefined professional layouts
  • Automatically resize and crop images to fit each layout
  • Apply consistent visual templates for color, spacing, and style
  • Export high-resolution, watermark-free images

What it does not do

  • No AI or machine learning
  • No cloud processing
  • No subscriptions
  • No image uploads

Why it exists

I built this because most modern collage tools are either subscription-based, cloud-dependent, or overloaded with features I didn’t need. This tool focuses on predictable layouts, local processing, and simple control.
